# Standard population

Standard population

## Usage

``` r
standardPopulation(name = "esp2013")
```

## Arguments

- name:

  Dataset to use. One of "esp2013", "espSex2013", "wsp2025" or
  "wspSex2025".

## Value

Tibble

## Examples

``` r
# \donttest{
standard_data <- standardPopulation("esp2013")
# }
```
